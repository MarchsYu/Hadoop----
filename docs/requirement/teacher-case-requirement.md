# 《酒店客房出租率分析案例》需求整理

## 老师案例核心要求

1. 数据源包含酒店基本信息、订单信息、客房信息。  
2. 使用 HDFS 保存原始数据和处理结果。  
3. 使用 MapReduce 完成清洗与统计分析。  
4. 统计酒店销售数量（或消费数量）排名。  
5. 统计各城市各酒店出租率。  
6. 统计各城市整体出租率。  
7. 使用 ECharts 展示分析结果。  
8. Sqoop、Web 采集先作为背景说明，当前阶段使用 CSV 模拟导入。

## 本项目当前实现范围

- 已完成目录结构升级、数据样例准备、3 个 MapReduce 任务代码骨架。  
- 重点保证在 Linux 虚拟机上通过命令行跑通。  
- 复杂业务逻辑与指标优化放在下一阶段迭代。
